ABOUT MIPSIM

MIPSIM was developed as a tool to simulate an embedded MIPS processor on a host linux desktop. It currently reads ELF32 little-endian MIPS-linux binaries and has been tested against many floating point and integer SPEC benchmarks. MIPSIM is free, open-source software under the GNU General Public License. See the enclosed documentation for details.

MIPSIM features:

  • Built-in debugger
    To debug, run emips with the '-d' flag. This will allow you to step through the program and dump instruction counters.
  • GDB Interface
    Although it is not fully tested, the emulator can interface with GDB and DDD for more advanced debugging.
